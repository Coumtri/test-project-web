Projet test web
=========================================

Web project source :
---------------------
https://github.com/kolorobot/spring-mvc-quickstart-archetype

Installation
------------

```bash
    git clone https://github.com/Coumtri/test-project-web.git
    mvn clean package
```

Run the project
----------------
mvn test tomcat7:run
OR 
java -jar target/dependency/webapp-runner.jar target/*.war
